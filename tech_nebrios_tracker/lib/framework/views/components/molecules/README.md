## ¿Qué es una Molécula en Atomic Design?

Convinaciones simples de Átomos que trabajan juntos para formar una funcionalidad.

### Ejemplos

- Búsqueda
- Alerta
- Botón con icono y texto
- Texto con etiquetas
- Mini perfil
